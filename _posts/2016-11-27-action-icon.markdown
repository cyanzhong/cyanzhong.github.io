# 制作标准
- 保持背景色为透明或者白色
- 图标内容为深色（例如黑色），尽量保证纯色
- 最佳图片尺寸：72px * 72px

# 推荐网站
- [Iconfactory](http://iconfactory.com/freeware)
- [Iconfont](http://www.iconfont.cn/plus)
- [Iconfinder](https://www.iconfinder.com/)
- [EasyIcon](http://www.easyicon.net/)
- [FlatIcon](http://www.flaticon.com/)
